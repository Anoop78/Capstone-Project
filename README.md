# 🎯 Career Guidance System

An intelligent web-based career prediction tool that helps students discover suitable career paths based on their academic performance, subject interests, analytical skills, and personality traits.

## 📋 Overview

The Career Guidance System analyzes multiple student parameters including Class 10th and 12th marks, favorite subjects, analytical aptitude, soft skills, technical interests, and work style preferences to generate **three personalized career recommendations**. Each recommendation includes a job title, detailed description, and confidence score.

## ✨ Features

- **Multi-factor Analysis**: Combines academic scores, subject interests, analytical ability, soft skills, technical inclination, and work style
- **Smart Career Mapping**: Rule-based prediction engine with weighted modifiers tailored to each career path
- **Three Career Outputs**: Always provides 3 distinct career recommendations sorted by match confidence
- **Real-time Validation**: Input validation ensures all values are within 0-100 range
- **Responsive Design**: Fully responsive UI that works on desktop, tablet, and mobile devices
- **Modern UI**: Glass-morphism design with smooth animations and visual confidence indicators

## 📊 Input Parameters

| Parameter | Description | Range |
|-----------|-------------|-------|
| Class 10th Marks | Percentage scored in 10th board exams | 0-100 |
| Class 12th Marks | Percentage scored in 12th board exams | 0-100 |
| Favorite Subject | Primary interest area (6 options) | Dropdown selection |
| Analytical Aptitude | Problem-solving and logical reasoning ability | 0-100 |
| Soft Skills | Communication, leadership, teamwork | 0-100 |
| Technical Interest | Interest in coding, tools, and technology | 0-100 |
| Work Style Preference | Collaborative, Independent, Creative, or Structured | Dropdown |

## 🧠 Subject Interest Options
